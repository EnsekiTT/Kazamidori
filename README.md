# Kazamidori

## 使い方
kazamidori.jsを読み込む

```javascript
positionDetect(1);
//id="show_arrow" に矢印表示
createArrow();
//位置情報の取得開始
startOrientationEvent();
```

友達の位置を受信する度に更新

```javascript
//緯度[deg]経度[deg]高度[m]
positionSet(35.692027, 139.701166,  37.0);
```
